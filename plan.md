Gemini 3 Flash와 ElevenLabs를 결합한 **'옵시디언 오디오 리뷰 자동화 시스템'**의 최종 구축 전략을 글로 정리해 드립니다. 시각적인 진행률 피드백과 Voice ID 최적화에 초점을 맞추었습니다.

---

### 1. 시스템 개요

본 시스템은 옵시디언의 텍스트 노트를 Gemini 3 Flash로 분석하여 개인화된 팟캐스트 스크립트를 생성하고, ElevenLabs의 고유 Voice ID를 통해 고품질 오디오로 변환한 뒤, 구글 드라이브를 거쳐 옵시디언 노트 내에 플레이어로 삽입하는 완전 자동화 워크플로우를 지향합니다.

### 2. 단계별 상세 실행 전략

**제1단계: Gemini 3 Flash를 통한 지능형 스크립트 생성**
먼저 사용자가 활성화한 노트를 분석합니다. Gemini 3 Flash 모델은 문서의 핵심 주제와 맥락을 파악하여 1인 내레이터용 스크립트를 작성합니다. 이때 단순 요약이 아니라 청중에게 말을 거는 듯한 구어체와 "이 부분은 꼭 기억하세요"와 같은 추임새를 포함하여 자연스러움을 극대화합니다. 특히 사용자가 설정한 ElevenLabs Voice ID의 성향(예: 차분한 지식인, 에너제틱한 강사 등)에 맞춰 톤앤매너를 조정하는 프롬프트를 적용합니다.

**제2단계: ElevenLabs Voice ID 기반 고품질 음성 합성**
추출된 스크립트를 ElevenLabs API에 전달합니다. 이때 미리 지정된 고유 Voice ID를 호출하여 사용자가 선호하는 일관된 목소리를 유지합니다. 'eleven_turbo_v2_5' 모델을 사용하여 생성 속도를 높이고, 안정적인 한국어 발화 환경을 구축합니다. 싱글 보이스 시스템을 통해 끊김 없는 안정적인 리뷰 콘텐츠를 생산합니다.

**제3단계: 실시간 진행 상황 시각화 (Progress UI)**
사용자 경험을 위해 옵시디언 내부에서 작업 진행률을 실시간으로 표시합니다. 실행 직후 "분석 중 - 스크립트 작성 중 - 음성 생성 중 - 클라우드 업로드 중"으로 이어지는 각 단계를 프로그레스 바(Progress Bar)나 상태 메시지 형태로 보여줍니다. 이는 API 통신 과정에서 발생할 수 있는 대기 시간 동안 사용자가 시스템의 작동 여부를 명확히 인지하게 만듭니다.

**제4단계: 구글 드라이브 클라우드 저장 및 권한 관리**
생성된 MP3 파일은 즉시 구글 드라이브의 지정된 폴더로 자동 업로드됩니다. 업로드와 동시에 API를 통해 파일 권한을 '링크가 있는 모든 사용자에게 공개'로 변경하여 외부 스트리밍이 가능한 상태로 만듭니다. 이후 웹에서 직접 재생 가능한 고유의 다이렉트 링크를 추출합니다.

**제5단계: 옵시디언 내 오디오 플레이어 자동 삽입**
추출된 링크를 바탕으로 옵시디언 노트 최상단에 HTML5 기반의 `<audio>` 태그를 자동으로 삽입합니다. 사용자는 별도의 플레이어를 실행할 필요 없이 옵시디언 안에서 재생, 일시정지, 볼륨 조절을 할 수 있으며, 세련된 디자인의 UI를 통해 노트를 읽으면서 동시에 리뷰를 청취하는 멀티모달 학습 환경을 경험하게 됩니다.

### 3. 기술적 기대 효과 및 장점

이 시스템을 구축하면 수동 작업 없이 단 한 번의 실행으로 개인 노트를 학습용 오디오 콘텐츠로 바꿀 수 있습니다. Gemini 3 Flash의 저렴하고 빠른 처리 능력과 ElevenLabs의 감성적인 음성 품질이 결합되어, 마치 나만을 위한 AI 튜터가 노트를 읽어주는 듯한 'NotebookLM' 수준의 경험을 제공합니다. 또한 향후 이미지나 도표 데이터까지 분석 범위에 포함할 수 있는 확장성을 보유하고 있습니다.
